- Review the [Two-Torial guide](https://two-torial.maimaidxprism.plus/games/sega/chunithm/x-verse-x/setup/) (specifically First Launch)
- Ensure you have 'No Encryption' patch enabled via the [web patcher](https://webpatcher.maimaidxprism.plus/)
- Ensure A001 option (bare minimum) is in your option folder and it's unpacked correctly and not nested
-# It should list as `option/A001/<content>`, NOT as `option/A001/A001/<content>`.
- Make sure the [Visual Studio C++ Redistributable](https://github.com/abbodi1406/vcredist/releases) and [DirectX Runtime](https://www.microsoft.com/en-us/download/details.aspx?id=8109) are installed correctly
-# (You must fully extract it into a folder AND install via the `dxsetup.exe` that it adds. Just extraction is NOT enough.)
- Ensure you have an `ICF1` file in your `amfs` folder, and if the game boots to the title that it shows the correct game version
- Ensure you do not have protected game executables (they must be unpacked) and they are the correct executables for your game version

Attaching your `segatools.ini` file **with the keychip section removed** and any notable errors or logs will improve assistance.
